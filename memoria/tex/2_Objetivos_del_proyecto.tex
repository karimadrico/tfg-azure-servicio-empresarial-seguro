\capitulo{2}{Objetivos del proyecto}

El objetivo principal de este Trabajo Fin de Grado es el desarrollo de un servicio empresarial seguro basado en tecnologías de computación en la nube utilizando la plataforma Microsoft Azure.

Para alcanzar este objetivo principal se han definido los siguientes objetivos específicos:

\begin{itemize}

\item Analizar las características y ventajas de las plataformas de computación en la nube en entornos empresariales.

\item Diseñar una arquitectura de sistema basada en servicios cloud que permita la gestión de información empresarial de forma segura.

\item Implementar una aplicación que permita gestionar documentos e incidencias dentro de un entorno empresarial.

\item Desplegar la aplicación en la plataforma Microsoft Azure utilizando servicios gestionados de la plataforma.

\item Implementar mecanismos de control de versiones, integración continua y gestión de tareas mediante herramientas como GitHub y GitHub Actions.

\item Analizar la calidad del código desarrollado utilizando herramientas de análisis estático como SonarQube.

\end{itemize}

Además de los objetivos técnicos, el proyecto también persigue objetivos formativos relacionados con la aplicación práctica de conocimientos adquiridos durante el grado en Ingeniería Informática, especialmente en áreas como ingeniería del software, arquitecturas cloud, seguridad informática y desarrollo de aplicaciones.
