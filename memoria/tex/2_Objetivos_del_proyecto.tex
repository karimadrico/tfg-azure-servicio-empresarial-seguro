\capitulo{2}{Objetivos del proyecto}

El objetivo principal del proyecto es desarrollar una aplicación empresarial segura desplegada en Microsoft Azure capaz de gestionar incidencias empresariales mediante una arquitectura cloud moderna.

Para alcanzar este objetivo principal se han definido los siguientes objetivos específicos:

\begin{itemize}
    \item Analizar las ventajas de las plataformas cloud en entornos empresariales.
    \item Diseñar una arquitectura segura basada en servicios cloud.
    \item Implementar una API REST utilizando Python y Flask.
    \item Desplegar la aplicación en Microsoft Azure.
    \item Gestionar secretos y configuraciones sensibles mediante Azure Key Vault.
    \item Implementar mecanismos de integración continua utilizando GitHub Actions.
    \item Gestionar el proyecto utilizando metodologías ágiles y control de versiones.
    \item Implementar una clasificación básica de incidencias utilizando técnicas sencillas de inteligencia artificial.
    \item Analizar la calidad del código utilizando SonarQube.
    \item Generar documentación técnica y funcional del sistema.
\end{itemize}

Además de los objetivos técnicos, el proyecto busca aplicar conocimientos adquiridos durante el Grado en Ingeniería Informática relacionados con ingeniería del software, seguridad informática, arquitecturas cloud y desarrollo backend.
