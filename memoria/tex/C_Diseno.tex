\section{Introducción}

Este anexo describe el diseño general del sistema desarrollado.

\section{Diseño arquitectónico}

La arquitectura sigue un modelo basado en servicios cloud desplegados sobre Microsoft Azure.

Los principales componentes son:

\begin{itemize}
    \item Azure App Service.
    \item API Flask.
    \item Azure Storage.
    \item Azure Key Vault.
    \item Azure Monitor.
\end{itemize}

\section{Diseño procedimental}

El flujo principal del sistema comienza cuando un usuario crea una incidencia mediante una petición HTTP.

La API valida la información recibida y almacena los datos utilizando servicios cloud gestionados.
