\apendice{Especificación de diseño}

\section{Introducción}

Este anexo describe el diseño del sistema desarrollado durante el proyecto.

\section{Diseño de datos}

El sistema almacena información relacionada con documentos empresariales e incidencias generadas durante el uso del sistema.

\section{Diseño arquitectónico}

La arquitectura del sistema se basa en el uso de servicios cloud proporcionados por Microsoft Azure.

Los principales componentes del sistema son:

\begin{itemize}

\item Servicio de aplicación encargado de procesar las solicitudes.

\item Servicio de almacenamiento para persistir la información.

\item Servicios de monitorización para supervisar el sistema.

\end{itemize}

\section{Diseño procedimental}

El sistema sigue un flujo de procesamiento en el que los usuarios realizan solicitudes al sistema, las cuales son procesadas por la aplicación y almacenadas en los servicios de almacenamiento correspondientes.
