\capitulo{3}{Conceptos teóricos}

\section{Cloud Computing}

El cloud computing es un modelo de computación que permite el acceso bajo demanda a recursos informáticos compartidos a través de Internet. Estos recursos pueden incluir servidores, almacenamiento, bases de datos, redes, software o servicios analíticos.

Las plataformas cloud permiten escalar recursos de forma dinámica, pagar únicamente por el uso realizado y reducir la complejidad de la gestión de infraestructuras físicas.

\section{Microsoft Azure}

Microsoft Azure es una plataforma de servicios cloud proporcionada por Microsoft que permite desarrollar, desplegar y gestionar aplicaciones en centros de datos distribuidos a nivel mundial.

Entre los servicios más relevantes que ofrece Azure se encuentran:

\begin{itemize}
\item Azure App Service
\item Azure Storage
\item Azure Monitor
\item Azure Virtual Machines
\end{itemize}

\section{Arquitectura basada en servicios}

Las arquitecturas basadas en servicios permiten estructurar aplicaciones en componentes independientes que se comunican entre sí mediante interfaces bien definidas.

Este tipo de arquitectura facilita la escalabilidad, el mantenimiento y la evolución de los sistemas software.

\section{DevOps}

DevOps es una metodología que integra el desarrollo de software y las operaciones de sistemas con el objetivo de mejorar la velocidad de entrega y la calidad del software.

Entre las prácticas habituales de DevOps se encuentran la integración continua, el despliegue continuo y la automatización de pruebas.
