\section{Introducción}

El desarrollo de soluciones cloud puede contribuir significativamente a mejorar la sostenibilidad tecnológica y organizativa.

En este proyecto se han aplicado diferentes principios relacionados con sostenibilidad tecnológica:

\begin{itemize}
    \item Uso eficiente de recursos cloud.
    \item Automatización de procesos.
    \item Reducción de infraestructuras físicas.
    \item Escalabilidad bajo demanda.
\end{itemize}

Además, la utilización de plataformas cloud gestionadas permite reducir el consumo energético asociado a infraestructuras locales tradicionales.

La automatización de despliegues y monitorización también mejora la eficiencia operativa y reduce errores humanos.

Desde un punto de vista formativo, este proyecto ha permitido desarrollar competencias relacionadas con sostenibilidad tecnológica, seguridad informática y desarrollo responsable de software.
