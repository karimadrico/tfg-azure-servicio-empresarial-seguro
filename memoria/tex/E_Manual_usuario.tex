\apendice{Documentación de usuario}

\section{Introducción}

Este documento describe cómo utilizar el sistema desarrollado durante el proyecto. El objetivo es proporcionar una guía básica que permita a los usuarios interactuar con la aplicación de forma sencilla.

El sistema desarrollado permite gestionar información empresarial mediante una plataforma desplegada en la nube utilizando Microsoft Azure.

\section{Requisitos de usuarios}

Para utilizar el sistema es necesario disponer de:

\begin{itemize}

\item Un navegador web (Google Chrome, Firefox, Edge, etc.)
\item Conexión a Internet
\item Acceso a la aplicación desplegada en la plataforma cloud
\end{itemize}

No es necesario instalar software adicional en el equipo del usuario.

\section{Instalación}

El sistema se encuentra desplegado en la plataforma Microsoft Azure, por lo que el usuario únicamente debe acceder a la dirección web donde se encuentra disponible la aplicación.

Una vez desplegado el sistema, el acceso se realiza mediante una URL proporcionada por el servicio de alojamiento.

\section{Manual del usuario}

El funcionamiento básico del sistema es el siguiente:

\begin{enumerate}

\item El usuario accede a la aplicación mediante su navegador web.

\item El sistema muestra la interfaz principal desde la cual se pueden consultar o gestionar los recursos disponibles.

\item El usuario puede realizar diferentes acciones dependiendo de las funcionalidades implementadas, como por ejemplo consultar información, registrar incidencias o visualizar métricas del sistema.

\item Las acciones realizadas por el usuario son procesadas por el sistema y almacenadas en los servicios cloud correspondientes.

\end{enumerate}

La interfaz del sistema ha sido diseñada con el objetivo de ofrecer una experiencia sencilla e intuitiva para los usuarios.

En caso de producirse algún error durante el uso del sistema, se mostrará un mensaje informativo que permitirá identificar el problema.
