\apendice{Documentación de usuario}

\section{Introducción}

Este anexo describe el funcionamiento básico de la aplicación desarrollada.

\section{Requisitos de usuarios}

El usuario únicamente necesita disponer de:

\begin{itemize}
    \item Navegador web.
    \item Conexión a Internet.
\end{itemize}

\section{Instalación}

La aplicación se encuentra desplegada en Microsoft Azure mediante Azure App Service.

\section{Manual del usuario}

El sistema permite:

\begin{itemize}
    \item Crear incidencias.
    \item Consultar incidencias.
    \item Consultar métricas básicas.
\end{itemize}

Ejemplo de creación de incidencia:

\begin{verbatim}
POST /incidencias
{
  "titulo": "Error servidor",
  "prioridad": "alta"
}
\end{verbatim}

Consulta de incidencias:

\begin{verbatim}
GET /incidencias
\end{verbatim}
