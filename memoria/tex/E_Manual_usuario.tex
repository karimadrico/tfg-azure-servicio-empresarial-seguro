\section{Operaciones disponibles}

\subsection{Consultar estado del servicio}

\begin{verbatim}
GET /
\end{verbatim}

\subsection{Crear solicitud desde el portal}

El usuario puede abrir el portal web, seleccionar el tipo de solicitud, introducir título, descripción y correo del solicitante, y enviar el formulario. La API devolverá un identificador, prioridad, clasificación y recomendación operativa.

\subsection{Crear solicitud desde API}

\begin{verbatim}
POST /solicitudes
Content-Type: application/json

{
  "tipo_solicitud": "acceso",
  "titulo": "Acceso VPN",
  "descripcion": "Necesito acceso VPN al entorno cloud",
  "reportado_por": "usuario@empresa.com",
  "prioridad": "media"
}
\end{verbatim}

El sistema clasificará automáticamente la solicitud y devolverá prioridad, categoría, tipo, nivel de confianza y recomendación.

\subsection{Consultar solicitudes}

\begin{verbatim}
GET /solicitudes?estado=abierta&prioridad=alta&tipo_solicitud=acceso
Authorization: Bearer <token>
\end{verbatim}

Si se accede a esta URL directamente desde el navegador sin cabecera de autorización, la API responde \texttt{\{"error":"No autenticado"\}}, que es el comportamiento esperado para proteger el listado.

\subsection{Consultar métricas}

\begin{verbatim}
GET /metricas
Authorization: Bearer <token>
\end{verbatim}
