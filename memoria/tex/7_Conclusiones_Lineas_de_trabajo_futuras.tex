\capitulo{7}{Conclusiones y Líneas de trabajo futuras}

El desarrollo de este Trabajo Fin de Grado ha permitido aplicar conocimientos relacionados con ingeniería del software, arquitecturas cloud, seguridad informática y automatización de procesos.

Uno de los principales resultados obtenidos ha sido el diseño e implementación de una arquitectura empresarial segura utilizando Microsoft Azure.

La solución desarrollada demuestra cómo es posible construir aplicaciones empresariales modernas utilizando servicios cloud gestionados y herramientas DevOps.

Durante el proyecto se han utilizado tecnologías empleadas en entornos empresariales reales: GitHub Actions, Terraform, Azure App Service, Azure Blob Storage, Azure Key Vault y Azure Logic Apps.

Además, el proyecto ha permitido profundizar en aspectos relacionados con:

\begin{itemize}
    \item Seguridad cloud.
    \item Gestión de secretos.
    \item APIs REST.
    \item Automatización.
    \item Arquitecturas escalables.
\end{itemize}

El prototipo desarrollado valida los objetivos del TFG: automatización de un proceso empresarial (gestión de incidencias), seguridad mediante Key Vault y Managed Identity, despliegue automatizado con CI/CD, e incorporación de un componente de análisis automático para clasificación de incidencias.

Como líneas futuras se plantean:

\begin{itemize}
    \item Sustituir el clasificador basado en reglas por un modelo de aprendizaje automático.
    \item Incorporar autenticación de usuarios con Azure AD.
    \item Añadir Application Insights y dashboards de monitorización.
    \item Desarrollar una interfaz web para usuarios no técnicos.
    \item Migrar la persistencia a Azure SQL o Cosmos DB para consultas más complejas.
\end{itemize}
