\capitulo{7}{Conclusiones y Líneas de trabajo futuras}

El desarrollo de este proyecto ha permitido aplicar de forma práctica diversos conocimientos adquiridos durante el grado en Ingeniería Informática, especialmente en áreas relacionadas con la ingeniería del software, las arquitecturas cloud y la gestión de proyectos software.

Uno de los principales resultados del trabajo ha sido el diseño e implementación de un servicio empresarial basado en tecnologías cloud utilizando la plataforma Microsoft Azure.

La solución desarrollada demuestra cómo es posible construir aplicaciones empresariales modernas utilizando servicios cloud gestionados, reduciendo la complejidad de la gestión de infraestructuras y facilitando la escalabilidad del sistema.

Además, el uso de herramientas de control de versiones, integración continua y análisis de calidad ha permitido aplicar buenas prácticas de desarrollo software durante todo el ciclo de vida del proyecto.

Como líneas de trabajo futuras se proponen las siguientes mejoras:

\begin{itemize}

\item Incorporar nuevos servicios cloud que permitan ampliar las funcionalidades del sistema.

\item Mejorar los mecanismos de monitorización y análisis del rendimiento.

\item Integrar servicios de inteligencia artificial para automatizar determinados procesos empresariales.

\item Implementar nuevas funcionalidades que permitan mejorar la experiencia de los usuarios del sistema.

\end{itemize}
