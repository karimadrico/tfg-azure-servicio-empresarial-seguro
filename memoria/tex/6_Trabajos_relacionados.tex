\capitulo{6}{Trabajos relacionados}

En los últimos años se han desarrollado numerosos proyectos relacionados con el uso de tecnologías cloud en entornos empresariales.

Uno de los trabajos más relevantes en este ámbito es el desarrollo de plataformas de servicios basadas en arquitecturas cloud que permiten gestionar aplicaciones empresariales de forma escalable.

Diversos trabajos académicos han analizado el uso de plataformas como Microsoft Azure, Amazon Web Services o Google Cloud para el desarrollo de aplicaciones distribuidas.

En el ámbito académico, diferentes Trabajos Fin de Grado han abordado el desarrollo de aplicaciones cloud para la gestión de información empresarial, la automatización de procesos o el análisis de datos.

El presente proyecto se diferencia de estos trabajos en que combina diferentes aspectos clave de la ingeniería del software, incluyendo el diseño arquitectónico, la seguridad, la automatización del despliegue y la monitorización del sistema.
