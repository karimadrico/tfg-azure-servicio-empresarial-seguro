\capitulo{1}{Introducción}

En los últimos años, el uso de tecnologías de computación en la nube se ha convertido en un elemento fundamental para el desarrollo de aplicaciones empresariales escalables, seguras y altamente disponibles. Las plataformas cloud permiten a las organizaciones desplegar infraestructuras tecnológicas sin necesidad de mantener hardware propio, reduciendo costes y aumentando la flexibilidad operativa.

Entre las diferentes plataformas cloud existentes, Microsoft Azure se posiciona como una de las soluciones más utilizadas en entornos empresariales, ofreciendo servicios para el desarrollo, despliegue y monitorización de aplicaciones de forma segura.

El objetivo de este Trabajo Fin de Grado es analizar, diseñar e implementar un servicio empresarial seguro basado en tecnologías cloud utilizando Microsoft Azure. El sistema desarrollado permitirá gestionar información empresarial mediante una arquitectura basada en servicios, integrando funcionalidades como almacenamiento de documentos, gestión de incidencias y monitorización mediante paneles de métricas.

Durante el desarrollo del proyecto se aplicarán metodologías ágiles de desarrollo software, utilizando herramientas de control de versiones, integración continua y seguimiento de tareas para garantizar la trazabilidad del proceso de desarrollo.
