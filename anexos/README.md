# Anexos técnicos

Los anexos oficiales del TFG se generan desde `memoria/anexos.tex` y se entregan como `memoria/anexos.pdf`.

Esta carpeta se mantiene como índice auxiliar para que el repositorio sea fácil de navegar. La documentación válida para entrega está organizada en:

| Bloque | Fuente principal |
|--------|------------------|
| Plan de proyecto | `memoria/tex/A_Plan_proyecto.tex` |
| Requisitos | `memoria/tex/B_Requisitos.tex` |
| Diseño | `memoria/tex/C_Diseno.tex` |
| Manual del programador | `memoria/tex/D_Manual_programador.tex` |
| Manual de usuario | `memoria/tex/E_Manual_usuario.tex` |
| Sostenibilidad | `memoria/tex/F_ODS.tex` |

Las capturas y evidencias complementarias se encuentran en `docs/`.
