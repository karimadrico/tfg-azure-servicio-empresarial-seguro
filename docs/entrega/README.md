# Documentación de entrega

Esta carpeta agrupa los documentos de apoyo para preparar la entrega final en UBUVirtual.

| Fichero | Uso |
|---------|-----|
| `checklist-tfg.md` | Lista de comprobación de entregables y reglas del TFG. |
| `enlaces-tfg.md` | Base del PDF final de enlaces solicitado por el tribunal. |

Antes de entregar, sustituir las URLs pendientes de los vídeos en `enlaces-tfg.md`, exportar ese contenido a PDF y subirlo junto a `memoria.pdf`, `anexos.pdf` y el código fuente.
