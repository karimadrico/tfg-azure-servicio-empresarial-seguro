# Problema empresarial

En muchas organizaciones la gestión de incidencias internas se realiza mediante correos electrónicos o sistemas poco estructurados.

Esto provoca varios problemas:

- Falta de trazabilidad de las incidencias
- Dificultad para priorizar problemas críticos
- Pérdida de información relevante
- Falta de métricas para mejorar los procesos de soporte

## Objetivo del proyecto

Desarrollar una solución cloud basada en Microsoft Azure que permita:

- Registrar incidencias empresariales
- Clasificarlas automáticamente mediante inteligencia artificial
- Almacenarlas de forma segura
- Monitorizar el sistema
- Proporcionar métricas sobre el servicio

La solución se desplegará en Azure utilizando App Service, Key Vault y Storage.
