## Descripción

Este proyecto desarrolla un servicio empresarial seguro en Microsoft Azure que permite la gestión de documentos, incidencias y métricas mediante arquitectura cloud escalable.

## Arquitectura

Azure App Service  
Azure Storage  
Azure Monitor  
Azure Dashboard

## Tecnologías

- Microsoft Azure
- GitHub Actions
- SonarQube
- Python / Node / Backend
